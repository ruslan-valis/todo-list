# Account
> Simple REST API to manage tasks creation / visualization (like a TODO list)

## Install

``` bash
$ docker-compose up
$ docker-compose exec app composer install
```

## Usage

``` bash

```

## Testing

``` bash
$ vendor/bin/phpunit
```

## Cleanup

``` bash
$ docker-compose down --rmi all
```
